# Unified pipeline

This folder contains the end to end entrypoint used for the reproducible FOX pangenome workflow.

## Entry points

- `run_unified_pipeline.sh` runs upstream scan, downloads proteins, clusters gene families, and runs modeling.
- `run_postprocess_09_12.sh` runs postprocessing scripts that generate tiered tables and manuscript figures.

See the repository root `README.md` and `docs/` for full installation and reproducibility guidance.
